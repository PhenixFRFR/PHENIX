# 🔥 PHENIX
## Autonomous Distributed Antenna System
### 自律分散アンテナシステム

---

## 📋 プロジェクト概要

PHENIXは、ラジコン母艦とドローンが連携しながら自律的に通信ノードを展開し、
平常時は既存インフラ（4G/5G/LTE/WiFi）を活用、
災害時はLoRaメッシュネットワークに自動切替することで、
インフラに依存しない自律型通信・センシング網を構築するシステムです。

---

## 🚀 主な特徴

- **自動ノード投下** - RSSI監視により電波弱化を検知し自動投下
- **デュアルモード通信** - 平常時4G/LTE、災害時LoRaに自動切替
- **三角測量システム** - サーマルカメラ3点で生存者を3D座標で特定
- **VTOL2機交互運用** - 常時空中監視を維持
- **分散レーダーネットワーク** - ノードが協調してセンシング

---

## 🛠️ システム構成

### ハードウェア
| コンポーネント | 仕様 |
|--------------|------|
| ラジコン母艦 | 1/8スケール 4×4オフロード |
| ドローン（試験機） | GEPRC MOZ7 V2 O4 Pro |
| VTOL（将来） | ArgusFPV Y3 × 2機 |
| 通信モジュール | SX1262 LoRa 915MHz |
| マイコン | ESP32-S3 + Radxa Zero 3W |
| GPS | NEO-M10Q |
| レーダー | LD2410 + LD2450 mmWave |
| UWB | DWM3000 |

### ソフトウェア
| ソフト | 用途 |
|--------|------|
| PHENIX Command Center | リアルタイム監視・制御 |
| LoRa Manager | 通信管理・SQLiteログ |
| QGroundControl | ドローン飛行制御 |
| ArduPilot | 自律飛行システム |
